# MoveXpress Logistics Website

## Project Overview

MoveXpress is a logistics company website developed using WordPress. The goal of this project was to design and build a modern, responsive website that presents a logistics company's services in a professional and user-friendly way.
The website was created using the WordPress Block Editor (Gutenberg) and customized with CSS to achieve the desired layout and styling.

## Development Process

The project began with planning the website structure and identifying the key pages required for a logistics company. These included the Home, About, Services and Tracking pages.

Using the Twenty Twenty-Five WordPress theme as a foundation, I designed each page with a focus on simplicity, consistency and responsiveness. Custom CSS was used to refine the layout, adjust spacing and improve the overall appearance where the default theme settings were not sufficient.

## Technologies Used

- WordPress
- Gutenberg Block Editor
- CSS
- Local by Flywheel
- Twenty Twenty-Five Theme

## Challenges

One of the main challenges during development was customizing the default spacing and layout provided by the Twenty Twenty-Five theme. This required additional CSS to achieve a cleaner design and improve the overall user experience.
